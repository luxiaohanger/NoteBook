## 圈复杂度

| 程序结构        |                              判定节点数 |
| ----------- | ---------------------------------: |
| `if`        |                               $+1$ |
| `if-else`   |                               $+1$ |
| `else if`   |                            每个 $+1$ |
| `for`       |                               $+1$ |
| `while`     |                               $+1$ |
| `do-while`  |                               $+1$ |
| 三目运算 `?:`   |                               $+1$ |
| `switch`    |                           分支数 $-1$ |
| `case`      |             参与 `switch` 分支数，不单独重复算 |
| `default`   |             参与 `switch` 分支数，不单独重复算 |
| `&&`、`\|\|` | 普通圈复杂度中不单独算；若按短路条件口径，每个基本条件可算 $+1$ |

公式：

$$
V(G)=判定节点数+1
$$

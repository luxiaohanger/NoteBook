## 设计类图

**设计类图**是在软件设计阶段使用的 UML 类图，用来描述系统中类的结构、属性、方法以及类之间的关系。

------
## 类的三栏结构

设计类图中，一个类通常分为三栏：**类名、属性、操作**。

```text
┌──────────────────────┐
│ 类名                  │
├──────────────────────┤
│ 属性                  │
├──────────────────────┤
│ 操作 / 方法            │
└──────────────────────┘
```

例如：

```text
┌────────────────────────────┐
│ Student                    │
├────────────────────────────┤
│ - id: Integer              │
│ - name: String             │
│ - major: String            │
├────────────────────────────┤
│ + getId(): Integer         │
│ + setName(name: String): void │
└────────────────────────────┘
```

符号含义：

```text
+ public，公有
- private，私有
# protected，受保护
~ package，包内可见
```

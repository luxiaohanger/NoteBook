## 传输介质特性总览

| 传输介质 | 信号形式 | 主要特点          | 抗干扰能力  | 传输距离 | 典型用途       |
| ---- | ---- | ------------- | ------ | ---- | ---------- |
| 双绞线  | 电信号  | 便宜、易安装、距离较短   | 一般     | 短    | 局域网、电话线    |
| 同轴电缆 | 电信号  | 屏蔽较好、带宽较高     | 较强     | 中等   | 早期以太网、有线电视 |
| 光纤   | 光信号  | 带宽极高、距离远、抗干扰强 | 极强     | 远    | 骨干网、长距离通信  |
| 微波   | 电磁波  | 无线传输、需要视距传播   | 易受环境影响 | 中远   | 卫星通信、无线链路  |

---

## 双绞线

双绞线由两根互相绝缘的铜导线按一定密度绞合而成，用来传输**电信号**。

绞合的作用是减少电磁干扰和串扰。

| 特性   | 说明                   |
| ---- | -------------------- |
| 成本   | 低                    |
| 安装   | 方便                   |
| 带宽   | 较低到中等                |
| 抗干扰  | 一般，屏蔽双绞线更好           |
| 距离   | 较短                   |
| 常见分类 | 非屏蔽双绞线 UTP、屏蔽双绞线 STP |


---

## 同轴电缆

同轴电缆由内导体、绝缘层、外导体屏蔽层和保护层组成，也传输**电信号**。

由于有屏蔽层，同轴电缆比双绞线抗干扰能力更强，带宽和传输距离也通常更好。

| 特性   | 说明            |
| ---- | ------------- |
| 成本   | 中等            |
| 带宽   | 较高            |
| 抗干扰  | 较强            |
| 距离   | 中等            |
| 结构特点 | 内外导体同轴，外层屏蔽   |
| 常见用途 | 有线电视、早期总线型以太网 |

---

## 光纤

光纤使用光脉冲在玻璃或塑料纤芯中传播，传输的是**光信号**。

光纤的核心优点是带宽极高、损耗低、传输距离远，并且不受电磁干扰。

| 特性   | 说明                |
| ---- | ----------------- |
| 成本   | 介质成本下降，但设备和施工成本较高 |
| 带宽   | 极高                |
| 抗干扰  | 极强，不受电磁干扰         |
| 距离   | 很远                |
| 安全性  | 较高，不易被窃听          |
| 常见分类 | 单模光纤、多模光纤         |

单模光纤适合长距离高速传输，多模光纤适合较短距离传输。

---

## 微波

微波属于无线传输介质，使用高频电磁波在空间中传播。

微波通信通常要求发送端和接收端之间尽量保持**视距传播**，即中间不能有明显障碍物。

| 特性   | 说明                    |
| ---- | --------------------- |
| 介质类型 | 非导引型介质                |
| 信号形式 | 电磁波                   |
| 安装   | 不需要铺设线缆               |
| 抗干扰  | 受天气、障碍物、电磁环境影响        |
| 距离   | 可中远距离传输               |
| 常见用途 | 地面微波通信、卫星通信、无线局域网部分场景 |

----

## 常用接入技术总览

| 接入技术              | 传输介质      | 典型场景       | 主要特点         |
| ----------------- | --------- | ---------- | ------------ |
| 电话拨号接入            | 电话线       | 早期家庭上网     | 速率低，占用电话线    |
| ADSL              | 电话线       | 早期宽带接入     | 下行速率高于上行速率   |
| HFC / Cable Modem | 同轴电缆 + 光纤 | 有线电视网宽带    | 共享带宽，覆盖家庭用户  |
| 光纤接入 FTTx         | 光纤        | 现代家庭、企业宽带  | 速率高、距离远、抗干扰强 |
| 以太网接入             | 双绞线 / 光纤  | 校园网、小区、企业  | 成本低、局域网技术成熟  |
| WLAN 无线局域网        | 无线电波      | Wi-Fi 上网   | 移动方便，易受干扰    |
| 蜂窝移动接入            | 无线电波      | 手机上网       | 覆盖广，支持移动性    |
| 卫星接入              | 微波 / 卫星链路 | 偏远地区、海上、航空 | 覆盖范围大，时延较高   |

## 电话拨号接入

电话拨号接入利用传统电话线，通过调制解调器把数字信号转换成模拟信号在电话网上传输。

| 特性     | 说明          |
| ------ | ----------- |
| 介质     | 双绞电话线       |
| 设备     | 调制解调器 Modem |
| 速率     | 很低          |
| 是否占用电话 | 通常占用电话线     |
| 典型特点   | 早期低速上网方式    |

> 拨号上网就是“用电话线打电话一样接入互联网”，速度慢，现在基本很少使用。

> 考试注意：拨号接入需要调制解调器，因为普通电话网传输的是模拟信号。

---

## ADSL 接入

ADSL 是非对称数字用户线，利用普通电话线传输数字信号。

“非对称”指的是下行速率和上行速率不同，通常：

$$
下行速率 > 上行速率
$$

| 特性       | 说明                                 |
| -------- | ---------------------------------- |
| 全称       | Asymmetric Digital Subscriber Line |
| 介质       | 普通电话线                              |
| 特点       | 上下行速率不对称                           |
| 是否可同时打电话 | 可以，通过频分复用分离语音和数据                   |
| 典型用途     | 早期家庭宽带                             |

> ADSL 适合普通家庭，因为用户通常下载多、上传少。

> 考试注意：ADSL 不等于拨号上网，它可以同时传输语音和数据。

---

## HFC / Cable Modem 接入

HFC 是混合光纤同轴网，通常由光纤和同轴电缆组成。

$$
HFC = 光纤 + 同轴电缆
$$

用户端通过 Cable Modem 接入互联网。

| 特性   | 说明                |
| ---- | ----------------- |
| 全称   | Hybrid Fiber Coax |
| 介质   | 主干光纤 + 用户侧同轴电缆    |
| 来源   | 有线电视网络改造          |
| 带宽特点 | 多用户共享带宽           |
| 典型用途 | 有线电视宽带接入          |

> HFC 可以理解为“用有线电视网络上网”。

> 考试注意：HFC 的带宽通常是共享的，同一区域用户多时可能相互影响。

---

## 光纤接入 FTTx

光纤接入是使用光纤把用户接入运营商网络，是现代宽带接入的重要方式。

FTTx 中的 x 表示光纤到达的位置。

| 类型   | 含义                    | 说明     |
| ---- | --------------------- | ------ |
| FTTH | Fiber To The Home     | 光纤到户   |
| FTTB | Fiber To The Building | 光纤到楼   |
| FTTC | Fiber To The Curb     | 光纤到路边  |
| FTTO | Fiber To The Office   | 光纤到办公室 |

| 特性   | 说明              |
| ---- | --------------- |
| 介质   | 光纤              |
| 带宽   | 高               |
| 传输距离 | 远               |
| 抗干扰  | 强               |
| 典型用途 | 家庭宽带、企业专线、校园网骨干 |

> 光纤接入就是把高速光纤尽量铺到离用户更近的位置。

> 考试注意：FTTH 是光纤到户，FTTB 是光纤到楼，不要混淆。

---

## 以太网接入

以太网接入使用以太网技术把用户接入局域网或城域网，再连接到互联网。

常见用户侧介质是双绞线，也可以使用光纤。

| 特性   | 说明            |
| ---- | ------------- |
| 介质   | 双绞线 / 光纤      |
| 技术基础 | Ethernet      |
| 设备   | 交换机、路由器       |
| 优点   | 成本低、部署方便、速率高  |
| 典型场景 | 校园网、小区宽带、企业网络 |

> 以太网接入就是把用户像接入局域网一样接入互联网。

> 考试注意：以太网接入常通过交换机实现，不是总线型旧以太网。

---

## WLAN 无线局域网接入

WLAN 使用无线电波实现局域范围内的无线接入，常见形式是 Wi-Fi。

| 特性   | 说明             |
| ---- | -------------- |
| 标准   | IEEE 802.11 系列 |
| 介质   | 无线电波           |
| 设备   | 无线路由器、AP、无线网卡  |
| 优点   | 移动方便、布线少       |
| 缺点   | 易受干扰，安全性需要加密保护 |
| 典型用途 | 家庭、校园、办公无线网络   |

> WLAN 就是平时说的 Wi-Fi 上网。

> 考试注意：WLAN 是局域网范围的无线接入，蜂窝移动网络是广域移动接入。

---

## 蜂窝移动接入

蜂窝移动接入通过基站和移动通信网络为用户提供互联网接入，例如 4G、5G。

| 特性   | 说明            |
| ---- | ------------- |
| 介质   | 无线电波          |
| 设备   | 手机、移动终端、基站    |
| 覆盖范围 | 广             |
| 移动性  | 强             |
| 典型技术 | 3G、4G、5G      |
| 典型用途 | 手机上网、移动热点、车联网 |

> 蜂窝接入就是手机通过基站上网。

> 考试注意：蜂窝网络强调广域覆盖和移动性，WLAN 强调局部区域高速无线接入。

---

## 卫星接入

卫星接入通过地面终端与通信卫星建立链路，再连接互联网。

| 特性   | 说明              |
| ---- | --------------- |
| 介质   | 微波 / 无线电波       |
| 覆盖范围 | 很大              |
| 优点   | 适合偏远地区          |
| 缺点   | 时延较高，成本较高，受天气影响 |
| 典型场景 | 海洋、山区、航空、应急通信   |

> 卫星接入适合“没有地面网络覆盖的地方”。

> 考试注意：卫星接入覆盖广，但传统卫星链路时延较大，不适合对时延特别敏感的应用。

## 传输时延与传播时延

| 类型   | 由什么决定       | 公式                      | 含义              |
| ---- | ----------- | ----------------------- | --------------- |
| 传输时延 | 分组长度、发送速率   | $d_{trans}=\frac{L}{R}$ | 把数据“推上链路”所需时间   |
| 传播时延 | 链路长度、信号传播速度 | $d_{prop}=\frac{D}{V}$  | 信号在链路中“跑过去”所需时间 |

传输：输入信道，因此时推送时间
传播：远距离播送，因此时路径耗时
